Detailed Description

Une structure struct DefineHuffmanCode (DHT) permet de stocker les informations lues dans une section DHT. Il faut donc en définir plusieurs pour stocker les différentes sections DHT.

Field Documentation

§ codes

struct HuffmanCode* codes

tableau de structures de struct HuffmanCode alloué au moment de la lecture de DHT avec countSymbols. Il faut penser à remplir les champs de chaque code : la connaissance du nombre de symboles par longueur de code et la lecture des valeurs/symboles qui suivent permettent de remplir les 2 champs de chaque code. La fonction buildHuffmanCodes remplit le champ code16.
